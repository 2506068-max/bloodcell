# Website Sistem Peredaran Darah
## Platform Pembelajaran Interaktif

### 📋 Deskripsi
Website edukatif komprehensif tentang sistem peredaran darah manusia yang dirancang untuk memudahkan pembelajaran dengan fitur-fitur interaktif, animasi, dan kuis.

### 📁 Struktur File

```
BLOOD CELL/
├── index.html                 (Halaman Utama / Home)
├── css/
│   └── style.css             (File CSS - Styling Website)
├── js/
│   └── script.js             (File JavaScript - Interaktivitas)
├── pages/
│   ├── pengertian.html       (1. Pengertian Sistem Peredaran Darah)
│   ├── organ.html            (2. Organ Peredaran Darah)
│   ├── jenis.html            (3. Jenis Peredaran Darah)
│   ├── gangguan.html         (4. Gangguan & Penyakit)
│   ├── galeri.html           (5. Galeri & Animasi)
│   ├── kuis.html             (6. Kuis Interaktif)
│   └── tentang.html          (7. Tentang Website)
└── README.md                 (File ini)
```

### 🚀 Cara Membuka Website

#### Metode 1: Membuka File HTML Langsung
1. Buka folder `BLOOD CELL`
2. Klik kanan pada file `index.html`
3. Pilih "Open with" → Pilih browser (Chrome, Firefox, Edge, Safari, dll)
4. Website akan terbuka di browser Anda

#### Metode 2: Menggunakan VS Code Live Server
1. Buka folder `BLOOD CELL` di VS Code
2. Install extension "Live Server" (jika belum ada)
3. Klik kanan pada `index.html` → Pilih "Open with Live Server"
4. Website akan otomatis terbuka di browser dengan hot reload

#### Metode 3: Menggunakan Local Server Python
```bash
# Buka terminal/command prompt di folder BLOOD CELL
# Untuk Python 3.x:
python -m http.server 8000

# Kemudian buka browser dan ketik:
http://localhost:8000
```

#### Metode 4: Menggunakan Live Server (Node.js)
```bash
# Install live-server (jika belum ada):
npm install -g live-server

# Buka terminal di folder BLOOD CELL dan jalankan:
live-server

# Browser akan otomatis membuka website
```

### 📚 Menu Pembelajaran

Website ini memiliki 7 menu pembelajaran utama:

1. **Pengertian Sistem Peredaran Darah**
   - Definisi dasar
   - Komponen utama
   - Fungsi sistem peredaran darah
   - Cara kerja sistem

2. **Organ Peredaran Darah**
   - Jantung (struktur, denyut, siklus)
   - Pembuluh Darah (Arteri, Vena, Kapiler)
   - Darah (Plasma, Sel Darah Merah, Sel Darah Putih, Trombosit)

3. **Jenis Peredaran Darah**
   - Peredaran Darah Kecil (Pulmonaris)
   - Peredaran Darah Besar (Sistemik)
   - Perbandingan kedua jenis

4. **Gangguan & Penyakit**
   - Hipertensi
   - Anemia
   - Penyakit Jantung

5. **Galeri & Animasi**
   - Animasi jantung berdenyut
   - Animasi aliran darah
   - Diagram sistem peredaran
   - Galeri informasi visual

6. **Kuis Interaktif**
   - 10 soal pilihan ganda
   - Evaluasi otomatis
   - Riwayat nilai

7. **Tentang Website**
   - Informasi website
   - Visi dan misi
   - Fitur-fitur
   - FAQ

### 🎯 Fitur Utama

✅ **Materi Lengkap** - Konten komprehensif tentang sistem peredaran darah
✅ **Visualisasi Interaktif** - Animasi dan diagram untuk pemahaman lebih baik
✅ **Kuis Interaktif** - 10 soal untuk menguji pemahaman
✅ **Responsif** - Dapat diakses dari desktop, tablet, dan smartphone
✅ **Navigasi Mudah** - Menu dropdown dan breadcrumb navigation
✅ **Riwayat Pembelajaran** - Menyimpan nilai kuis di local storage

### 💡 Tips Penggunaan

1. **Pelajari Secara Berurutan** - Mulai dari Pengertian hingga Tentang Website
2. **Baca dengan Teliti** - Pahami setiap konsep sebelum melanjutkan
3. **Lihat Animasi** - Gunakan visualisasi untuk pemahaman yang lebih baik
4. **Kerjakan Kuis** - Uji pemahaman Anda dengan kuis interaktif
5. **Ulangi Jika Diperlukan** - Belajar berkali-kali untuk hasil optimal

### 🔧 Teknologi yang Digunakan

- **HTML5** - Struktur halaman
- **CSS3** - Styling dan layout responsif
- **JavaScript** - Interaktivitas dan animasi
- **SVG** - Diagram dan grafis vektor
- **LocalStorage** - Penyimpanan riwayat nilai

### 📱 Kompatibilitas Browser

- ✅ Google Chrome
- ✅ Mozilla Firefox
- ✅ Microsoft Edge
- ✅ Safari
- ✅ Opera
- ✅ Brave

### 🎓 Untuk Guru/Pendidik

Website ini dapat digunakan untuk:
- Materi pembelajaran di kelas

- Tugas rumah interaktif
- Persiapan ujian
- Review materi sebelum ulangan

- Periksa file `pages/tentang.html` untuk informasi kontak
- Kirimkan feedback atau laporan bug
